How to Contribute

UniCalc is built and maintained by students. If you have verified closing merits for any university, campus, or program, sharing them helps us keep the data accurate for everyone. If you know the exact aggregate formula or test pattern for a university we have not covered yet, we would love to add it. And if you are a developer, the entire codebase is open source on GitHub. You can fix bugs, add features, or improve the UI directly.

Reach out at hi@azanw.com if you want to contribute in any way.
